CSDigitalFinds Buying Guide Generator

Upload these files to the root of your GitHub repository:
- buying-guide-generator.html
- guide-generator.js
- style.css

Then open:
https://csdigitalfinds-cell.github.io/Csdigitalfinds/buying-guide-generator.html

How to use:
1. Enter guide title.
2. Choose category.
3. Paste Amazon affiliate link.
4. Click Generate Buying Guide.
5. Download the HTML page.
6. Upload that generated HTML page to GitHub root.
7. Add it to sitemap.xml later.

This is the starter version. It generates static SEO-friendly buying guide pages.
